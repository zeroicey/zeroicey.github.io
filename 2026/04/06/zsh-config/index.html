常用模块

  • fast-syntax-highlighting - 语法高亮 (比 zsh-syntax-highlighting 更快更强大)
  • fzf-tab - 用 fzf 替代 zsh 默认补全菜单
  • zsh-autosuggestions - 命令历史自动补全
  • fzf - 模糊搜索
  • zoxide - 智能 cd
  • input - 键盘绑定和 vim 输入模式

💡 fzf-tab 需要在 compinit 之后、zsh-autosuggestions 之前加载
