窗口操作

在 tmux 中，每个会话可以有多个窗口。

快捷键 功能
Ctrl+b c 新建窗口
Ctrl+b , 重命名当前窗口
Ctrl+b n 切换到下一个窗口
Ctrl+b p 切换到上一个窗口
Ctrl+b 0-9 切换到指定编号的窗口
Ctrl+b w 列出所有窗口并选择
Ctrl+b & 关闭当前窗口

面板操作

Tmux 允许在一个窗口中分割成多个面板。

快捷键 功能
Ctrl+b % 垂直分割（左右）
Ctrl+b " 水平分割（上下）
Ctrl+b o 在面板间切换
Ctrl+b ; 切换到上一个面板
Ctrl+b 方向键 切换到指定方向的面板
Ctrl+b x 关闭当前面板
Ctrl+b z 最大化/还原当前面板
Ctrl+b { 将当前面板移到左边
Ctrl+b } 将当前面板移到右边
Ctrl+b Space 切换面板布局

复制模式

快捷键 功能

v 开始选择
y 复制选中内容
q 退出复制模式

小技巧

  1. 常用快捷键前缀: 记住 Ctrl+b 是所有快捷键的前缀
  2. 保持连接: 使用 tmux 即使 SSH 断开，任务也不会中断
  3. 会话命名: 给会话起个有意义的名字，方便管理
  4. 鼠标支持: 开启鼠标支持后可以直接用鼠标选择和调整面板

掌握这些命令，你就能高效地使用 tmux 进行终端管理了！
